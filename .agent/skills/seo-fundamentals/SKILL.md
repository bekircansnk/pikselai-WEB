# SEO Temelleri

> Arama motoru görünürlüğü için prensipler.

---

## 1. E-E-A-T Çerçevesi

| Prensip | Sinyaller |
|---------|-----------|
| **Deneyim (Experience)** | Birinci elden bilgi, gerçek örnekler |
| **Uzmanlık (Expertise)** | Kimlik bilgileri, bilgi derinliği |
| **Otorite (Authoritativeness)** | Backlinkler, bahsetmeler, endüstri tanınırlığı |
| **Güvenilirlik (Trustworthiness)** | HTTPS, şeffaflık, doğru bilgi |

---

## 2. Core Web Vitals

| Metrik | Hedef | Ölçtüğü |
|--------|-------|---------|
| **LCP** | < 2.5s | Yükleme performansı |
| **INP** | < 200ms | Etkileşim |
| **CLS** | < 0.1 | Görsel kararlılık |

---

## 3. Teknik SEO Prensipleri

### Site Yapısı

| Öğe | Amaç |
|-----|------|
| XML site haritası | Taramaya yardımcı ol |
| robots.txt | Erişimi kontrol et |
| Canonical etiketler | Kopyaları önle |
| HTTPS | Güvenlik sinyali |

### Performans

| Faktör | Etki |
|--------|------|
| Sayfa hızı | Core Web Vital |
| Mobil uyumluluk | Sıralama faktörü |
| Temiz URL'ler | Taranabilirlik |

---

## 4. İçerik SEO Prensipleri

### Sayfa Öğeleri

| Öğe | En İyi Uygulama |
|-----|-----------------|
| Title etiketi | 50-60 karakter, anahtar kelime önde |
| Meta açıklama | 150-160 karakter, ilgi çekici |
| H1 | Sayfa başına bir tane, ana anahtar kelime |
| H2-H6 | Mantıksal hiyerarşi |
| Alt metin | Açıklayıcı, anahtar kelime doldurulmamış |

### İçerik Kalitesi

| Faktör | Önem |
|--------|------|
| Derinlik | Kapsamlı kapsam |
| Tazelik | Düzenli güncellemeler |
| Benzersizlik | Orijinal değer |
| Okunabilirlik | Net yazım |

## 6. YZ İçerik Yönergeleri

### Google Ne Arar

| ✅ Yap | ❌ Yapma |
|--------|----------|
| YZ taslağı + insan düzenlemesi | Ham YZ içeriği yayınlama |
| Orijinal içgörüler ekle | Değersiz kopyala |
| Uzman incelemesi | Gerçek kontrolünü (fact-check) atlama |
| E-E-A-T izle | Anahtar kelime doldurma |
